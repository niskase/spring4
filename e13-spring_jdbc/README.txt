SISÄLTÖ

* Spring JDBC

TOIMINTA
--------
HenkilolistaHandler pyytää dao-luokalta listallisen henkilöitä ja tulostaa niiden etunimet konsoliin.
Tämän jälkeen handler pyytää dao-luokalta yksittäistä henkilöä ja tulostaa sen tiedot konsoliin.
HenkilollistaHandler lisää vielä lopuksi uuden henkilön tietokantaan dao-luokan avulla ja tulostaa
lopuksi kaikki henkilöt konsoliin.
